Home work for Helmes by Alexander Karakin

### Installation:
```
1) git clone https://github.com/BotServicePro/taskForHelmes.git
2) cd taskForHelmes
3) composer update
Ensure that you have configured DB connection on .env file
4) make migrate
5) make seed
6) make start
7) php artisan key:generate
8) go to route /show
```
